desain & fitur

trevelia prototype

halaman login

Ini adalah menu dimana user diharuskan untuk login atau mendaftarkan diri nya ke aplikasi , apabila user sudah pernah mendaftar maka ia hanya butuh untuk memasukkan email dan password. Dan apabila user belum memiliki akun pada aplikasi ini, maka ia dapat mengklik tombol daftar dan akan diarahkan ke halaman pendaftaran akun.

trevelia prototype

halaman utama

Setelah user login/mendaftarkan akun maka mereka akan diarahkan ke menu utama. Disini ada beberapa fitur dan menu yang dapat dilihat oleh user seperti menu pencarian, menu pemesanan tiket hotel, pesawat, kereta, ataupun mencari sebuah restaurant dari aplikasi ini. User juga dapat melakukan edit profile Ketika ia mengklik foto profil yang berada pada pojok kiri atas. Dari beranda anda juga dapat berpindah menu seperti ke menu promo, pesanan ataupun customer service yang berada pada bagian bawah aplikasi ini.

trevelia prototype

halaman destinasi

Pada halaman ini diperlihatkan beberapa pilihan dan referensi yang disediakan apliaksi untuk user. Seperti pada gambar diperlihatkan beberapa pilihan hotel favorit yang sering dikunjungi.

trevelia prototype

halaman pemesanan

Ini adalah menu pemesanan dimana didalamnya user dapat melihat pesanan yang sedang mereka lakukan dimana pesanan ini menunggu finalisasi dan pembayaran dari user, desain yang digunakan cukup simpel agar para user tidak merasa ribet saat melakukan transaksi.
